# WTA_project
ONLINE CINEMA RESERVATION MANAGEMENT SYSTEM
Entertainment is a very crucial part of oneâ€™s individual life. The common and slickest way of refreshment can be watching a movie. Online movie booking system is a web portal where you can book tickets in advance, know your movie show timing, watch movie trailers and read reviews for the same. IMDb movie database serves as a useful repository of movie reviews which can be accessed and the trailers are also made available for the customers. The objective of Cinema Reservation System is to provide the facility of booking movie tickets online. It is a PHP/MySQL based client-server application where customers can view timing of movie shows and book the show as per the availability. This project provides a ticket reservation system allowing bookings in a few easy steps. Users can easily book for Shows, view all the shows available and make payment accordingly.
